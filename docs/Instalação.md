# Instalação

## Pré-requisitos
- BepInEx-IL2CPP
- .NET 6.0 SDK

## Passos de instalação
1. Clonar o repositório e abrir a solução
2. Incluir o mod na pasta de mods
3. Gerar os assemblies com o build apropriado

## Como testar
1. Executar o ambiente com o mod carregado
2. Pressionar Ctrl+P para verificar a ativação do export
